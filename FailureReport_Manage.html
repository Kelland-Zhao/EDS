此页面用于管理故障报告是否需要书写的判定。

This page is used for managing whether failure reports need to be written or not.

注塑工序(IM)故障时间大于240min，原则需要书写故障报告。

IM process failure time greater than 240min, failure report should be written in principle.

植磨毛工序(TF)故障时间大于120min，原则需要书写故障报告。

TF process failure time greater than 120min, failure report should be written in principle.

包装工序(PK)故障时间大于60min，原则需要书写故障报告。

PK process failure time greater than 60min, failure report should be written in principle.

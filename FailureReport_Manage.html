故障报告管理功能/ Failure Report Management Functions

此页面用于管理故障报告是否需要书写的判定。

This page is used for managing whether failure reports need to be written or not.

注塑工序(IM)故障时间大于240min，原则需要书写故障报告。

IM process failure time greater than 240min, failure report should be written in principle.

植磨毛工序(TF)故障时间大于120min，原则需要书写故障报告。

TF process failure time greater than 120min, failure report should be written in principle.

包装工序(PK)故障时间大于60min，原则需要书写故障报告。

PK process failure time greater than 60min, failure report should be written in principle.

手动添加使用说明 / Manual Add Instructions

  • 点击右上角“手动添加 / Manual Add”按钮后，会在当前工序表格末尾生成一条黄色高亮的手动录入行。 After clicking the “手动添加 / Manual Add” button, a highlighted manual entry row is appended to the current process table.
  • 系统自动生成编号，维修时间字段仅支持数字输入，其余字段按照提示完成填写。 The report number is auto-generated; the repair time field only accepts digits, and other fields follow the on-screen placeholders.
  • 信息填写完毕后，请使用“需要 / 不需要故障报告”按钮完成判定。 After filling in the details, complete the decision using the “需要故障报告 / Need Failure Report” or “不需要故障报告 / No Failure Report” buttons.

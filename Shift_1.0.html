故障记录/ Failure Record

填写说明 / Instructions
  • 需要填写本班欠产大于半小时的机台情况 / Record machines with production shortage exceeding 30 minutes in current shift
  • 需要填写重点质量问题的跟进情况 / Record follow-up on key quality issues:
    • 注塑：工艺参数异常 / Injection Molding: Process parameter abnormalities
    • 植磨毛：无金属丝 / Flocking: No metal wire issues
  • 故障处理过程填写方法 / How to fill in Failure Process:
    1. 找到维修该故障时对应的维修内容（如：主机/ 自动化/ 工艺等）/ Identify the related maintenance category (e.g., Main machine / Automation / Process, etc.)
    2. 将对应选项（是/否）选为“是”/ Select “Yes” for the corresponding option
    3. 在后续输入框中输入具体维修内容 / Enter detailed maintenance actions in the follow-up input box
    提示 / Tips:
    • 默认所有部件都选“是”，请根据实际情况选择 / All items are defaulted to “Yes”, please adjust based on actual situation
    • 一旦选择“是”，后续输入框不可为空，否则无法提交 / If “Yes” is selected, the follow-up input cannot be empty, otherwise submission will be blocked
    • 直接原因 / Direct Cause 与 建议措施 / Suggested Action 为必填项，否则无法提交 / Direct Cause and Suggested Action are required fields, submission will be blocked if empty
数据源说明 / Data Source Information

处理说明 / Follow-up Instructions
  • 重大故障判定标准：注塑（IM）> 240分钟、植磨毛（TF）> 120分钟、包装（PK）> 60分钟 / Major failure criteria: IM > 240 mins, TF > 120 mins, PK > 60 mins
  • 重大故障需要在每日SDM讨论，并决定后续措施（培训/ 改善/ 保养）/ Major failures must be discussed in daily SDM and determine follow-up actions (Training / Kaizen/CI / PM)
  • 非重大故障由工序技术组长判定后续措施 / Non-major failures are determined by process technical team leader for follow-up actions
  • 已解决和未解决故障都需要遵循上述规则 / Both solved and unsolved failures must follow the above rules
  • 直接原因 / Direct Cause 与 建议措施 / Suggested Action 为必填项，否则无法提交 / Direct Cause and Suggested Action are required fields, submission will be blocked if empty
显示范围说明 / Display Range Information
  • 未解决故障 / Unsolved Failures：显示所有未解决的故障记录，无时间限制 / Display all unsolved failure records without time limitation
  • 已解决故障 / Solved Failures：仅显示最近30天内解决的故障记录 / Only display failure records resolved within the last 30 days
  • 历史数据 / Historical Data：30天前已解决的故障记录不会显示，以优化页面性能 / Failure records resolved more than 30 days ago are not displayed to optimize page performance
  • 数据来源 / Data Source：包含所有工序（注塑/植磨毛/包装）的故障记录 / Includes failure records from all processes (Injection/Flocking/Packaging)

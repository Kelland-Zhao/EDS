三班转保养跟进 / PM Shift Follow-up
使用说明 / Instructions

工序切换 / Process Switching: 点击上方Tab切换不同工序的数据（IM/TF/PK）/ Click the tabs above to switch between different process data (IM/TF/PK)

状态编辑 / Status Editing: 单击状态列可选择Done或Ongoing状态 / Click on the Status column to select Done or Ongoing status

反馈编辑 / Feedback Editing: 双击"后续措施 - 保养（反馈）"列可输入跟进信息 / Double-click on the "Follow-up Action - PM (Feedback)" column to enter follow-up information

批量提交 / Batch Submission: 编辑完成后点击右上角"提交修改"按钮保存所有更改 / After editing, click the "Submit Changes" button in the top-right corner to save all changes

数据源说明 / Data Source Information

数据来源 / Data Source: 数据来源于MasterData表格，包含所有工序的保养跟进记录 / Data comes from the MasterData sheet, containing PM follow-up records for all processes

更新机制 / Update Mechanism: 修改数据后需要点击提交按钮才能保存到MasterData / After modifying data, click the submit button to save to MasterData

实时同步 / Real-time Sync: 提交成功后会自动刷新数据，确保显示最新信息 / After successful submission, data will be automatically refreshed to ensure the latest information is displayed

状态说明 / Status Description: Done表示已完成，Ongoing表示进行中 / Done indicates completed, Ongoing indicates in progress

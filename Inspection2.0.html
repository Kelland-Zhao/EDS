点检2.0 / Inspection 2.0

点检任务筛选/ Inspection Task Filter

使用说明 / Instructions
  • 机台主数据维护 / Machine Master Data Maintenance: 一旦涉及到任何机台增加，请各工序管理员在主数据MachineList sheet中维护主数据 / Once any machine is added, please have each process manager maintain the master data in the MachineList sheet
  • 点检执行人需要手动输入工号，姓名会自动显示 / Inspector needs to manually enter employee ID, name will be displayed automatically
  • 班次需要自行选择 / Shift needs to be selected manually
  • 任务参数选择顺序 / Task Parameters Selection Order: 点检人类型 → 任务类型 → 机台号 → 车间 → 工序 / Inspector Type → Task Type → Workcenter → Workshop → Process
  • 机台号需要手动选择（有红色备注的是RBM到期未及时完成的点检机台）/ Workcenter needs to be selected manually (those with red notes are RBM overdue inspection machines)
  • 其余信息会基于已选择的参数自动联动显示 / Other information will be automatically filtered based on selected parameters
数据源说明 / Data Source Information
  • 任务类型来自于Tasklist MoC状态生效的任务清单，见Tasklist_history文件 / Task types are sourced from task lists with "Effective" MoC status in the Tasklist_history file
